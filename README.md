# 轻记 - 个人财务记账应用

移动端优先的个人财务记账单页应用，使用 localStorage 存储数据，通过 Chart.js 实现统计图表。

## 快速开始

直接在浏览器中打开 `index.html` 即可运行，无需构建。

## 功能特点

- 📝 快速记账 - 支持 10 种消费分类
- 📊 统计报表 - 月度支出环形图 + 近6月趋势折线图
- 📱 移动端优先 - 适配手机浏览器
- 🔐 数据隔离 - 多用户账号系统
- 💾 本地存储 - 无需后端，数据存储在本地

## 页面结构

- `#/login` - 登录页
- `#/register` - 注册页
- `#/` - 首页（今日概览 + 最近记录）
- `#/add` - 记账页
- `#/records` - 账单列表
- `#/stats` - 统计页
- `#/me` - 我的页面
